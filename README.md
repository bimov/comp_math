Графики получены запуском с такими параметрами:

```
python3 main.py --ticker AAPL --start 2017-01-01 --end 2019-12-31 \
  --maturity 2020-01-31 --r 0.02 --n-s 60 --n-t 40 --strike-policy median
```

Ticker: Котировка

Start: Дата началы замеров

End: Даты окончания замеров

Maturity: Даты экспирации T

r: Безрисковая ставка

n_s: Число узлов сетки по цене S

n_t: Число узлов сетки по времени t

Strike-policy: Правило выбора страйка K
